# simple_mips_processor_for_fpga
Implementation of mips processor for fpga with assembler for it

# Build assembler
<code> make build </code>

# Usage
<code> ./ram_maker mips_test.asm ram.data </code>

# Supported mips instructions
- [x] add
- [x] addi
- [x] sub
- [x] or
- [x] and
- [x] lw
- [x] sw
- [x] beq
- [x] slt
- [x] j
